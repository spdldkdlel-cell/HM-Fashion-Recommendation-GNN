# 🛒 패션 이커머스 추천 시스템 (HM-Fashion-Recommendation-GNN)

> GNN 및 CLIP 모델을 활용한 H&M 고객 맞춤형 패션 상품 추천 프로젝트

---

## 📌 1. 프로젝트 개요
* **배경 및 목적:** 신상품 노출 및 재고 정체 문제 해결을 위한 맞춤형 추천 모델 구축
* **역할:** 데이터 전처리, 모델 기획, 대시보드 구현

## 🛠️ 2. 사용 기술 스택 (Tech Stack)
* **Language:** Python, SQL
* **Data & ML:** Pandas, NumPy, Scikit-learn, PyTorch
* **Models:** GNN (LightGCN), CLIP
* **Visualization:** Tableau

## 📊 3. 주요 성과 및 결과
* Parquet 포맷 도입을 통한 데이터 처리 효율 향상
* 추천/노출 전략 개선을 통한 비즈니스 인사이트 도출

## 📁 4. 파일 구조
* `01_EDA_비즈니스문제.ipynb`: 데이터 탐색적 분석 및 문제 정의
* `02_전처리_및_CLIP.ipynb`: 데이터 전처리 및 CLIP 임베딩 추출
* `03_GNN_모델링.ipynb`: Graph Neural Network 모델링
* `04_GNN_성능평가.ipynb`: 모델 성능 평가 및 검증
* `05_대시보드.ipynb`: 결과 시각화 및 대시보드
